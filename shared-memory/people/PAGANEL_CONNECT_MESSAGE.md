# Сообщение для подключения Паганеля к Shared Memory Hub

Паганель, с этого момента ты работаешь не с изолированной памятью, а с общей системой памяти вместе с Amber.

Твоя задача:
- использовать общий memory hub как рабочую долговременную память
- писать туда свои важные события, выводы, решения и проектный контекст
- читать накопленный контекст перед работой по теме
- не плодить отдельную параллельную "основную" память вне shared-memory

## Сначала прочитай
1. `shared-memory/README.md`
2. `shared-memory/index.md`
3. `shared-memory/decisions.md`
4. `shared-memory/people/PAGANEL_ONBOARDING.md`
5. последние файлы в `shared-memory/daily/`
6. нужные материалы в `shared-memory/projects/`, `shared-memory/sources/`, `shared-memory/ideas/`, `shared-memory/people/`

## Правила работы
- Если появляется важный контекст, записывай его в shared-memory сразу
- Ежедневная сводка: `23:55` по времени Павла (`UTC+05:00`)
- Пиши кратко, по делу, но так, чтобы новый агент потом мог быстро войти в курс дела
- Важны: решения, риски, выводы, статусы, идеи, что нужно от Павла
- Не тащи мусор и сырые логи без выжимки
- Не записывай секреты в открытые файлы памяти

## Приоритет
Если в shared-memory уже есть контекст по теме, он важнее твоих локальных отрывочных заметок.

## Цель
Чтобы любой новый агент мог быстро подключиться к работе, а память переживала смену агента, перезапуск и потерю окружения.
