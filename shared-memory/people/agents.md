# Agents

## Amber
- Основной помощник Павла
- Ведёт общую память, решения, выжимки и координацию

## Paganel
- Второй агент
- Должен использовать общий memory hub, а не изолированную память

## Правило масштабирования
Если подключается третий агент:
- он не создаёт отдельную систему памяти с нуля
- он читает `shared-memory/README.md`, `index.md`, `decisions.md`, последние daily notes и нужные project/source files
- затем пишет в ту же общую структуру по тем же правилам
